InformationView

class InformationView : UIView

A view to show the general information about a set of seedlings.

It contains a chart on the top showing the rate of germination and death of the seedlings. This is followed by labels showing the sowing date, number of seeds sown, and counters for germination and death. There are many tap targets and a lot of user interaction.

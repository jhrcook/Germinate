Germination Tracker


(In development)

A simple iOS app to track the germination of my seeds.

Checkout a summary of the app on the project page on my website.

The following GIF prevides a general summary of the current app.

Third-Party Libraries

  • SnapKit for setting layout constraints.
  • Chameleon for some theming (for lower than iOS 13).
  • KeyboardObserver for handling the notifications from the keyboard.
  • SwiftyButton provides a nice, flat button.

I made the icon using Procreate](https://procreate.art) and formatted it for iOS using MakeAppIcon.
